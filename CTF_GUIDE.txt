## üìù Challenge Add Karne Ka Tareeka

### Multiple Images Format (Recommended):

```javascript
{
  id: 2,
  title: "Challenge Ka Naam",           // Challenge ka title
  category: "Web Exploitation",         // Category: Web, Priv Esc, Crypto, Forensics, etc
  difficulty: "Easy",                   // Easy | Medium | Hard | Insane
  platform: "TryHackMe",                // TryHackMe | HackTheBox | CTFtime
  tags: ["Tag1", "Tag2", "Tag3"],       // Max 3-4 tags
  images: [
    "/images/ctf/photo1.png",           // Photo 1
    "/images/ctf/photo2.png",           // Photo 2
    "/images/ctf/photo3.png"            // Photo 3
  ],
  completedDate: "2024-01-20",          // Format: YYYY-MM-DD
  description: "Is challenge mein aapne kya sikhya aur kaise kiya. Thodi si lambi description likhein.",
  walkthroughEnabled: true,             // true = walkthrough dikhega, false = nahi
  walkthrough: {
    steps: [
      {
        title: "Step 1: Reconnaissance",    // Step ka title
        description: "Kya kiya step mein",  // Step ka description
        code: "nmap -sV 10.10.10.10"        // Command ya code
      },
      {
        title: "Step 2: Exploitation",
        description: "Kaise kiya exploitation",
        code: "exploit command here"
      }
    ],
    tools: ["nmap", "gobuster", "burp"],    // Tools jo use kiye
    learnings: ["Lesson 1", "Lesson 2"]      // Kya sikhya
  }
},
```

### Single Image Format:

```javascript
{
  id: 3,
  title: "Simple Challenge",
  category: "Cryptography",
  difficulty: "Medium",
  platform: "HackTheBox",
  tags: ["Crypto", "RSA"],
  image: "/images/ctf/my-challenge.png",    // Single image
  completedDate: "2024-02-15",
  description: "Challenge ka short description.",
  walkthroughEnabled: false,
  walkthrough: null
},
```

## ‚ö†Ô∏è Important Notes

1. **ID Unique Hona Chahiye:** Har challenge ki unique id ho
2. **Date Format:** YYYY-MM-DD (e.g., 2024-01-20)
3. **Images Path:** `public/images/ctf/` folder mein honi chahiye
4. **Comma Dhyan Se:** Last item ke baad comma nahi, baaki sabke baad comma
